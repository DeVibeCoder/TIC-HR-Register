# HR System Frontend

A React + Vite frontend starter for an HR system dashboard.

## Tech Stack

- React 19
- Vite 8
- TypeScript

## Quick Start

1. Install dependencies:
   npm install
2. Start development server:
   npm run dev
3. Build for production:
   npm run build
4. Preview production build:
   npm run preview

## Project Notes

- Entry point: `src/main.tsx`
- Main screen: `src/App.tsx`
- Styling: `src/styles.css`

The current UI includes a starter HR dashboard with metrics and an employee table so development can begin immediately.
